# UI / UX Standard

Interfaces should match the product's purpose and audience.

- Operational tools should be calm, dense, and scannable.
- Games can be more expressive and playful.
- Avoid decorative complexity that hurts usability.
- Do not use landing pages when the user asked for a usable app, game, or tool.
- Verify responsive layouts before shipping.
- Websites and web apps must be designed for both phone-sized and desktop-sized screens by default.
- Desktop app layouts should avoid assumptions that only work on one operating system unless approved.

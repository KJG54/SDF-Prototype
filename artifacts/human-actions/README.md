# Human Actions

This folder stores framework-level human-action artifacts.

Use it when the Software Factory framework itself needs the human to do something outside normal conversation, such as installing a tool, authenticating, running a manual command, testing something locally, or making an external decision.

Active project human actions belong inside the project instead:

```text
projects/<slug>/artifacts/human-actions/
```

Use the paired template files:

```text
templates/artifacts/ACTION-001-human-action.md
templates/artifacts/ACTION-001-human-action.json
```

Current status: no open framework-level human actions are recorded here.

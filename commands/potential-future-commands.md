# Potential Future Commands

These are command ideas, not implemented slash commands and not approved automation scope.

## Proposed

| Prompt | Reserved Command | Status | Notes |
| --- | --- | --- | --- |
| `clean up files` | `/clean-up` | proposed | Structured cleanup review for files that may be unused, stale, generated, or confusing. Manual cleanup workflow exists in `standards/cleanup-workflow.md`; automation remains deferred. |
| `triage ideas` | `/triage-ideas` | documented | Added as a plain-language workflow prompt for classifying batches of rough ideas. Future automation may add durable intake artifacts. |
| `route this idea` | `/route-idea` | documented | Added as a plain-language workflow prompt for classifying one idea and recommending a destination. |
| `create Claude delegation` | `/delegate` | documented | Create a bounded delegation packet for Claude Code. |
| `review Claude handoff` | `/review-handoff` | documented | Inspect Claude's result before decisions or integration. |
| `split work between Codex and Claude` | `/split-work` | documented | Recommend a token-conscious collaboration plan without changing approval boundaries. |

# Tool Registry

This registry tracks tools, MCP servers, plugins, skills, and services that may help Software Factory projects.

Statuses:

- `known`
- `preferred`
- `approved`
- `restricted`
- `deprecated`

The registry is a map, not permission to use every tool automatically.

# Phase 10: Memory / Lessons Learned

## Purpose

Record what should help future projects.

## Agent Responsibilities

- Create a final project memory packet.
- Propose global memory promotions for reusable lessons.
- Keep project memory separate from global memory.
- Update `PROJECT-CHECKLIST.md` and `project-checklist.json`.
- Ask approval before promoting durable lessons or preferences.

## Human Responsibilities

- Approve or reject proposed global memory updates.

## Required Output

- Project memory packet
- Optional global memory proposals
- Phase gate
- Final phase summary

## Exit Criteria

Lessons are recorded in the right place and the project can be referenced later.

# Phase 4: Build Plan

## Purpose

Create a complete plan agents can execute from scaffold to ship.

## Agent Responsibilities

- Break work into clear tasks.
- Include likely files or areas involved.
- Add acceptance criteria and verification steps.
- Mark Claude delegation candidates.
- Identify approval gates.
- Update `PROJECT-CHECKLIST.md` and `project-checklist.json`.

## Human Responsibilities

- Confirm the plan matches the desired project.
- Approve the build direction.

## Required Output

- `PLAN-001-build-plan.json`
- `PLAN-001-build-plan.md`
- Phase gate
- Phase summary

## Exit Criteria

The human approves the build plan.

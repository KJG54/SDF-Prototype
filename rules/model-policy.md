# Model Policy

V1 documents model policy without automating model routing.

## Capability Tiers

- `fast-low-cost`: summaries, formatting, routine validation, low-risk checks
- `balanced`: default project work
- `high-reasoning`: architecture, debugging, security, unclear requirements, project-shaping decisions
- `specialist-tool-backed`: tasks requiring specific tools, docs, or domain workflows

Use stronger models when the cost is justified by risk, ambiguity, or complexity.

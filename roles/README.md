# Roles

Roles are reusable hats that Codex or Claude may wear during a task. They are not separate agent identities.

## Initial Roles

- Architect
- Builder
- QA Reviewer
- Security Reviewer
- UX Reviewer
- Documentation Writer
- DevOps / Release
- Memory Curator

Agents may switch roles when helpful, but should say so when the role affects recommendations or review criteria.

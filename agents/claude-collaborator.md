# Claude Collaborator

Claude Code is an additional collaborator, not the lead.

## Best Uses

- Building larger code sections from a clear plan.
- Reviewing code, architecture, or tests.
- Debugging difficult issues.
- Checking Codex's work for gaps.
- Completing parallel tasks when the project is large enough.

## Boundaries

Claude must not change architecture, expand scope, add major dependencies, create external services, or make project-shaping decisions without approval through Codex and the human.
